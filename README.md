# 🌱 Ser Sustentável - Rede Social

Uma plataforma de rede social dedicada a promover práticas sustentáveis e conectar pessoas interessadas em um estilo de vida mais consciente com o meio ambiente.

## 🌐 Acesse a Aplicação

🔗 **Deploy ao vivo:** [https://ser-sustentavel-rede-social-frontend.pages.dev/](https://ser-sustentavel-rede-social-frontend.pages.dev/)

*Hospedado em Cloudflare Pages*

## 🎯 Sobre o Projeto

**Ser Sustentável** é uma rede social moderna que permite aos usuários:
- Compartilhar dicas e práticas sustentáveis
- Conectar com pessoas que têm os mesmos valores ambientais
- Inspirar outros a adotarem hábitos mais ecológicos
- Criar uma comunidade global consciosa

## 🛠️ Tecnologias

- **HTML5** - Estrutura semântica das páginas
- **CSS3** - Estilização responsiva e design moderno
- **JavaScript** - Interatividade e lógica da aplicação

**Distribuição de linguagens no projeto:**
- JavaScript
- HTML
- CSS
✨ Funcionalidades

    👥 Perfis de usuários personalizados
    📝 Criar e compartilhar posts sobre sustentabilidade
    ❤️ Curtir e comentar em publicações
    🔍 Explorar conteúdo de outros usuários
    💬 Conectar com pessoas que compartilham interesses
    🏷️ Utilizar hashtags para categorizar posts
    📱 Design responsivo para mobile e desktop
